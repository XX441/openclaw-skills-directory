# 🤖 agent-team-framework

**Category:** System  
**Folder:** `agent-team-framework`

## What it does

Reusable framework for managing multi-agent coding teams with clear roles, shared knowledge, quality gates, deployment flow, and governance.

## When to use it

Use when setting up or auditing a project that needs coordinated AI coding agents.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R agent-team-framework ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
