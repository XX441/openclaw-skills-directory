# 📊 ai-ppt-generator

**Category:** Design  
**Folder:** `ai-ppt-generator`

## What it does

Generate branded presentation decks from structured prompts, brand assets, and slide assembly scripts.

## When to use it

Use when creating a new deck with a repeatable brand-driven slide workflow.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ai-ppt-generator ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
