# 🎨 canva

**Category:** Design  
**Folder:** `canva-connect`

## What it does

Manage Canva designs, assets, exports, and folders through the Canva Connect API.

## When to use it

Use when automating Canva asset management, export, or design organization.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R canva-connect ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

- `SKILL.md`
- `references/api.md`
- `scripts/canva.sh`
