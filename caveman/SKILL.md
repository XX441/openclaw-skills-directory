Explain the concept in the simplest possible terms.

## Rules

1. No jargon. If a technical term is unavoidable, define it in plain language immediately.
2. Use concrete analogies from everyday life.
3. One concept per sentence.
4. Assume zero prior knowledge.
5. If the concept needs multiple layers, build up from the simplest layer first.
6. End with a one-sentence summary a child could understand.

## Format

- Start with the bottom-line in one sentence
- Use bullet points for the building blocks
- End with an analogy that ties it together
