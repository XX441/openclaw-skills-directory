# 🧑‍💻 coding-discipline

**Category:** Coding  
**Folder:** `coding-discipline`

## What it does

Applies disciplined coding rules before, during, and after code changes: think first, keep changes surgical, verify before done.

## When to use it

Use before any code change, refactor, feature, deploy, or bug fix.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R coding-discipline ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
