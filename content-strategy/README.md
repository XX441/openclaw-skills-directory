# 📝 content-strategy

**Category:** Business  
**Folder:** `content-strategy`

## What it does

Build content strategy, pillars, calendars, distribution plans, repurposing workflows, and performance metrics.

## When to use it

Use when planning what content to create, where to distribute it, and how to measure it.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R content-strategy ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
