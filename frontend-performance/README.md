# ⚡ frontend-performance

**Category:** Coding  
**Folder:** `frontend-performance`

## What it does

Analyze and improve frontend performance across Core Web Vitals, loading, bundle size, stability, and runtime efficiency.

## When to use it

Use when pages feel slow, bundles are heavy, or Core Web Vitals need improvement.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R frontend-performance ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
