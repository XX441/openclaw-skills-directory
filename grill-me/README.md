# 🔥 grill-me

**Category:** Strategy  
**Folder:** `grill-me`

## What it does

Relentlessly interviews and stress-tests plans, strategies, or designs until assumptions and dependencies are clear.

## When to use it

Use when you want a plan challenged, interrogated, or pressure-tested.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R grill-me ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
