# 📚 grill-with-docs

**Category:** Strategy  
**Folder:** `grill-with-docs`

## What it does

Stress-tests plans or interface designs against domain documentation and updates docs inline.

## When to use it

Use when specs, terminology, or architecture must stay aligned with a domain model.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R grill-with-docs ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
