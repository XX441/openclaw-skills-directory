# 🤝 handoff

**Category:** Workflow  
**Folder:** `handoff`

## What it does

Compacts current work into a clear handoff document for another agent or future session.

## When to use it

Use when work needs to continue elsewhere without losing context.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R handoff ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
