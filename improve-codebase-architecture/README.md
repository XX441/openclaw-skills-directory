# 🏗️ improve-codebase-architecture

**Category:** Coding  
**Folder:** `improve-codebase-architecture`

## What it does

Finds architecture deepening opportunities: shallow modules, tight coupling, untestable code, and AI-navigation issues.

## When to use it

Use when auditing or planning refactors for maintainability and testability.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R improve-codebase-architecture ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
