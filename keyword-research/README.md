# 🔎 keyword-research

**Category:** Marketing  
**Folder:** `keyword-research`

## What it does

SEO keyword research and topic discovery with intent classification, clustering, prioritization, and report templates.

## When to use it

Use when planning search-led content, topic clusters, or keyword strategy.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R keyword-research ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

## Files included

- `SKILL.md`
- `references/example-report.md`
- `references/keyword-intent-taxonomy.md`
- `references/keyword-prioritization-framework.md`
- `references/topic-cluster-templates.md`
