# 💼 linkedin-cli

**Category:** Business  
**Folder:** `linkedin-cli`

## What it does

CLI utilities for LinkedIn profile search, feed summaries, message checks, and session-based workflows.

## When to use it

Use when automating LinkedIn scouting or lightweight relationship intelligence.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R linkedin-cli ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
