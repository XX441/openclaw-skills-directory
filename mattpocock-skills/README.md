# 🛠️ mattpocock-skills

**Category:** Workflow  
**Folder:** `mattpocock-skills`

## What it does

Collection index of reusable engineering and thinking skills adapted from Matt Pocock-style agent workflows.

## When to use it

Use as a gateway/reference for related skills such as TDD, handoff, PRD, triage, and grilling.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R mattpocock-skills ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
