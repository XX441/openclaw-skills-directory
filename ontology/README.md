# 🕸️ ontology

**Category:** System  
**Folder:** `ontology`

## What it does

Typed knowledge graph pattern for structured memory, entities, relationships, constraints, and cross-skill state.

## When to use it

Use when information needs to be linked, queried, validated, or shared across skills.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ontology ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

## Files included

- `SKILL.md`
- `references/queries.md`
- `references/schema.md`
- `scripts/ontology.py`
