# ⚛️ react-expert

**Category:** Coding  
**Folder:** `react-expert`

## What it does

React 18+ / 19 guidance for component architecture, hooks, state, Server Components, Suspense, performance, and testing.

## When to use it

Use when building or reviewing modern React applications.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R react-expert ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
