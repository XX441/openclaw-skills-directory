# 🌱 self-improvement

**Category:** System  
**Folder:** `self-improving-agent`

## What it does

Captures mistakes, corrections, failures, feature requests, and reusable lessons for continuous agent improvement.

## When to use it

Use after failures, user corrections, tool/API issues, or recurring workflow discoveries.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R self-improving-agent ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
