# 🧭 ssot-knowledge-graph

**Category:** Knowledge  
**Folder:** `ssot-knowledge-graph`

## What it does

Connects SSOT files to projects, trackers, status docs, content indexes, and memory pointers.

## When to use it

Use when new knowledge needs to be classified, cross-referenced, and connected.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ssot-knowledge-graph ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

## Files included

- `SKILL.md`
- `references/connection-layer.md`
