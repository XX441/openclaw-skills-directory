# ⚙️ ssot-setting

**Category:** System  
**Folder:** `ssot-setting`

## What it does

Snapshots, backs up, and restores OpenClaw configuration as an SSOT-controlled workflow.

## When to use it

Use before/after config changes or when restoring a known-good configuration state.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ssot-setting ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

## Files included

- `SKILL.md`
- `scripts/ssot-restore.sh`
- `scripts/ssot-setting.sh`
