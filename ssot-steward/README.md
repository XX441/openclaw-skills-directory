# 🗂️ ssot-steward

**Category:** Knowledge  
**Folder:** `ssot-steward`

## What it does

Enforces SSOT file naming, versioning, archive, upload, and branded-PDF workflow rules.

## When to use it

Use for Google Drive / Good Steward file operations, version bumps, uploads, and PDF generation.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ssot-steward ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
