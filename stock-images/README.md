# 🖼️ Stock Images

**Category:** Design  
**Folder:** `stock-images`

## What it does

Finds free stock photos and placeholder image sources using direct URL patterns and provider guidance.

## When to use it

Use when sourcing visual placeholders or royalty-free stock imagery.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R stock-images ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
