# 🎫 to-issues

**Category:** Product  
**Folder:** `to-issues`

## What it does

Turns plans, PRDs, or conversations into prioritized, independently completable issue tickets.

## When to use it

Use when a strategy or spec needs actionable implementation tasks.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R to-issues ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
