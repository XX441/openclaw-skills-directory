# 📄 to-prd

**Category:** Product  
**Folder:** `to-prd`

## What it does

Synthesizes conversation context into a structured Product Requirements Document.

## When to use it

Use when requirements need to be documented as a PRD/spec.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R to-prd ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.

- `SKILL.md`
