# 🚦 triage

**Category:** Workflow  
**Folder:** `triage`

## What it does

Sorts incoming messages, files, emails, or ideas into prioritized action buckets.

## When to use it

Use when processing a batch of inputs or deciding what matters first.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R triage ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
