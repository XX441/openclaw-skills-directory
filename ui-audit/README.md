# 🔍 ui-audit

**Category:** Design  
**Folder:** `ui-audit`

## What it does

Audits UI against UX principles: hierarchy, accessibility, cognition, navigation, feedback, and usability patterns.

## When to use it

Use when reviewing an interface or design for UX quality.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ui-audit ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
