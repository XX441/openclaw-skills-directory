# ✨ ui-ux-pro-max

**Category:** Design  
**Folder:** `ui-ux`

## What it does

Searchable UI/UX design databases and scripts for styles, palettes, typography, charts, stacks, and design systems.

## When to use it

Use when generating design direction or choosing UI patterns from structured references.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R ui-ux ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
