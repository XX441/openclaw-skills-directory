# 🌦️ weather

**Category:** Utility  
**Folder:** `weather`

## What it does

Gets current weather and forecasts from no-key public weather providers.

## When to use it

Use when a task needs quick weather context.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R weather ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
