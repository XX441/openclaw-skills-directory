# 🔭 zoom-out

**Category:** Strategy  
**Folder:** `zoom-out`

## What it does

Steps back from details to identify larger patterns, strategic connections, assumptions, and implications.

## When to use it

Use when you need big-picture perspective or “what am I missing?” analysis.

## How to install

Copy this folder into your OpenClaw workspace skills directory:

```bash
cp -R zoom-out ~/.openclaw/workspace/skills/
```

Then restart or refresh your OpenClaw skill registry if needed.
